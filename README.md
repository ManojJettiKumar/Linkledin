# LinkedIn Clone

A modern LinkedIn clone built with React, TypeScript, and Firebase. This project demonstrates a social networking platform with core LinkedIn features.

## Tech Stack

- React.js with TypeScript
- Firebase (Authentication & Database)
- Tailwind CSS for styling
- Vite as build tool

## Features

- User authentication (Sign up, Login, Forgot password)
- Profile management
- News feed with posts and comments
- Real-time messaging
- Network connections
- Job listings
- Dark/Light mode
- Responsive design

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/subha2002666/linkedin_clone.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

## Environment Setup

Create a `.env` file in the root directory with your Firebase configuration:

```
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
